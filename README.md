# Информационная система сдачи в аренду торговых площадей

Курсовой проект по дисциплине «Базы данных».

## Описание проекта

Проект представляет собой информационную систему для организации сдачи в аренду торговых площадей торгового центра.

Система позволяет:
- вести учет клиентов;
- вести учет торговых площадей;
- оформлять договоры аренды;
- привязывать к одному договору несколько торговых точек;
- учитывать платежи арендаторов;
- формировать финансовый отчет;
- рассчитывать аренду по дням;
- работать с ролями пользователей;
- использовать авторизацию и регистрацию клиентов.

## Технологии

- C++
- Qt
- PostgreSQL
- Qt SQL
- TCP-клиент/сервер
- MVC-подход
- Git

## Основные роли

### Администратор

Администратор может:
- управлять справочниками;
- оформлять договоры аренды;
- добавлять платежи;
- просматривать финансовый отчет;
- работать с клиентами, пользователями и торговыми площадями.

### Клиент

Клиент может:
- зарегистрироваться;
- войти в личный кабинет;
- просматривать свободные торговые площади;
- просматривать свои договоры;
- просматривать свои платежи;
- изменять регистрационные данные;
- менять пароль.
